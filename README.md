# AIVA# 🤖 AIVA - Artificial Intelligence Virtual Assistant

# 

# AIVA is an AI-powered virtual assistant built using Python, Streamlit, Ollama, LangChain, ChromaDB, and Stable Diffusion.

# 

# \## Features

# 

# \* 💬 AI Chat Assistant

# \* ✍️ AI Content Writer

# \* 📄 PDF Chat with RAG

# \* 🎨 AI Image Generation

# \* 🌐 Web Search Integration

# \* 📅 Productivity Tools

# \* ⚡ Desktop Automation

# \* 🧠 Persistent Memory System

# 

# \## Technology Stack

# 

# \* Python

# \* Streamlit

# \* Ollama

# \* LangChain

# \* ChromaDB

# \* SQLite

# \* Stable Diffusion
