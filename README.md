\# 🤖 AIVA - Artificial Intelligence Virtual Assistant



AIVA is an AI-powered virtual assistant built using Python, Streamlit, Ollama, LangChain, ChromaDB, and Stable Diffusion.



\## Features



\* AI Chat Assistant

\* AI Content Writer

\* PDF Chat (RAG)

\* AI Image Generator

\* Real-Time Web Search

\* Productivity Dashboard

\* Desktop Automation

\* Persistent Memory System



\## Technology Stack



\* Python

\* Streamlit

\* Ollama

\* LangChain

\* ChromaDB

\* SQLite

\* Stable Diffusion
